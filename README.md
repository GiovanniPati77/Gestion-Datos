# Gestion De Datos
Proyecto Gestion de Datos aplicando un CRUD en base de datos ( MySql )
* El usuario crea un cliente nuevo
* Se muestra el cliente en la lista 
* Puede modificar sus datos 
* Eliminar clientes 
* Todo conectado a una base de datos
